# Mochi Mochi Krater (もちもち)

A web-based monster battling game.

## Project Structure

- `index.html` - Main entry point
- `battle.html` - Battle screen
- `team.html` - Team management
- `assets/` - Images, sounds, etc.
- `css/` - Stylesheets
- `js/` - JavaScript logic
  - `monsters.js` - Monster data and functions
  - `moves.js` - Move definitions
  - `supabase.js` - Supabase integration

## Development

To run locally, simply open `index.html` in a web browser.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Supabase (for backend)
