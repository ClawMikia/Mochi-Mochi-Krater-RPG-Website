# Mochi Mochi Krater (もちもち)

A web-based monster battling game inspired by classics like Pokemon Deluge and Pokemon Crater.

## Project Structure

- `index.html` - Main entry point / monster catalog
- `battle.html` - Battle arena (1v1, 2v2, 3v3)
- `battle.html` - Team management / team builder
- `how-to-play.html` - How to play guide
- `css/styles.css` - Main stylesheet
- `js/monsters.js` - Monster data (200 monsters with stats 250-300)
- `js/moves.js` - Move definitions
- `js/game.js` - Core game logic
- `js/supabase.js` - Supabase integration
- `js/app.js` - App initialization and UI

## Features

- 200 comedy-named monsters with unique stats
- Type-based effectiveness system (Super Effective / Not Very Effective)
- 1v1, 2v2, 3v3 battle formats
- Player vs CPU & online matchmaking via Supabase

## Development

To run locally, simply open any HTML file (`index.html`, `battle.html`, `battle.html`, or `how-to-play.html`) in a web browser.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Supabase (for backend)
